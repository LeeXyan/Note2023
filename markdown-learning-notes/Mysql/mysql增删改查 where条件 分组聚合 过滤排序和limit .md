### 重命名

---

当我们进行select的一张表的时候，加入我们使用如下语法

```mysq
select 字段 as 新字段 ... from 表
# 同样也等于
select 字段 新字段 ... from 表
```

这样就给这个字段临时起了一个新的名字，但是请记住，是临时的！
